Conceptos
Clave

Un pequeño glosario para que todos hablemos el mismo idioma durante la charla.

Glosario · Conceptos clave
LLM

Large Language Model

Modelo de IA entrenado con grandes cantidades de texto para entender y generar lenguaje, responder preguntas y escribir código.

MCP

Model Context Protocol

Protocolo que permite a una IA conectarse de forma estandarizada con herramientas, APIs y bases de datos.

Skills

Instrucciones reutilizables

Enseñan a la IA cómo realizar una tarea específica de manera consistente.

CLI

Command Line Interface

Interfaz para interactuar con un programa mediante comandos escritos en la terminal.

Plugins

Extensiones de IA

Agregan nuevas funciones o integraciones a una herramienta de IA.

SRS

Software Requirements Spec.

Documento que define los requisitos funcionales y no funcionales del software.

PRD

Product Requirements Doc.

Describe qué producto se va a construir, para quién y qué problema resuelve.

Arquitectura

Documento Técnico

Explica cómo se implementará el sistema: tecnologías, componentes y decisiones técnicas.

La Seguridad no es Negociable

Vulnerabilidades comunes

Credenciales hardcodeadas (ej. "admin/123456") o controles de acceso rotos que exponen URLs internas sin login.

Nuestro rol

Security by Design y Security by Default: auditar cada sugerencia de la IA, hashing de contraseñas y validaciones robustas en el servidor.
